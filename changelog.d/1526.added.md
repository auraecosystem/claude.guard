- The GitHub App integration now supports **multiple organizations**. `gh-app setup` stores every installation of your App (not just one), and each launch auto-selects the installation covering the current repo's owner — so an agent working across repos in different orgs (or personal accounts) mints a correctly-scoped token for each without reconfiguration. `gh-app token`/`verify` gain `--org <login>` to target a specific installation, and `gh-app export`/`import` carry the full installation set between hosts. The token cache is keyed by owner too, so two same-named repos in different orgs can never be handed each other's token.
