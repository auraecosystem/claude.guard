- `claude-github-app create` (and `setup`) now creates your GitHub App in one click via GitHub's App-manifest flow: it opens a loopback page that hands the App's settings to GitHub and takes back the App ID + private key automatically — no form to fill, permissions to set, PEM to download, or App ID to paste. The hand walkthrough remains as an automatic fallback for headless/SSH hosts where a browser can't reach the loopback callback.
